# Atividade 3

Um hub desenvolvido para reunir todos os exercícios e mini projetos criados durante os estudos de JavaScript, Fetch API e manipulação de arquivos JSON.

O projeto possui uma identidade visual moderna utilizando as cores roxo e amarelo, funcionando como uma central de navegação para acessar cada exercício individualmente.

---

# 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* JSON
* Fetch API

---

# 🎨 Tema Visual

O visual do projeto foi inspirado em uma interface moderna com:

* Tons de roxo
* Destaques amarelos
* Cards interativos
* Efeitos hover
* Layout responsivo
* Navegação centralizada

---

# 📂 Projetos Disponíveis

## 👨‍🎓 Exercício 1 — Expansão da Base de Dados

Projeto responsável por ampliar o arquivo `alunos.json`, adicionando novos alunos e mantendo a estrutura original dos objetos.

### Funcionalidades:

* Manipulação de JSON
* Estrutura de arrays de objetos
* Organização de dados

---

## 📧 Exercício 2 — Novas Propriedades e Renderização

Evolução do sistema de alunos adicionando:

* cidade
* email

As novas propriedades passaram a ser exibidas dinamicamente nos cards utilizando JavaScript.

### Conceitos aplicados:

* Renderização dinâmica
* innerHTML
* Template Literals
* Manipulação de DOM

---

## 🎬 Exercício 3 — Catálogo de Filmes

Sistema desenvolvido para listar filmes carregados através de um arquivo `filmes.json`.

### Estrutura:

* nome
* nota
* gênero

### Funcionalidades:

* Leitura de JSON
* Exibição dinâmica
* Organização em cards

---

## 🛒 Exercício 4 — E-commerce de Produtos

Simulação de uma loja virtual criada do zero.

### Funcionalidades:

* Catálogo de produtos
* Carrinho de compras
* Cards animados
* Renderização via Fetch API
* Interface moderna

### Conceitos utilizados:

* Arrays de objetos
* Eventos
* Manipulação de DOM
* Estruturas dinâmicas

---

## 📚 Exercício 5 — Painel Interativo de Cursos

Projeto mais avançado utilizando:

* Objetos aninhados
* Arrays internos
* Filtros por categoria
* Renderização dinâmica

### Funcionalidades:

* Filtro com `<select>`
* Cursos separados por categoria
* Exibição de módulos
* Aviso de vagas encerradas
* Interface interativa

### Conceitos avançados:

* `.map()`
* `.join()`
* `.filter()`
* `fetch()`
* Objetos aninhados
